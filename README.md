# Otvoreno Računarstvo - 3. laboratorijska vježba

API koji izlaže bazu podataka programskih jezika.

### Pokretanje

Za pokrenuti API:
```
go run ./main.go
```

Za podignuti DB koristeći docker-compose:
```
cd db
docker-compose up
```
